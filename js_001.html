자바스크립트란?

다양한 객체들로 구성되어 있고, 메서드를 통해 제어하는 객채기반언어
메서드:명령어

객체와 메서드의 의미

스마트폰,전화해라(매개변수) → 스마트폰: 객체, 전화해라: 메서드 (매개변수): 정보

주요단어

  • 변수: 변화하는 값 / 선언문을 통해 제어된다 var.let.const
    const는 고정변수이기때문에 한번 입력하면 바꿀 수 없다
  • 함수: 계산식을 만듦 function(){ 계산식 }
  • 조건문: if문: 만일 ~~~한다면 ~~~이여라 if,else,else if
    else문: 만일~~~아니라면 ~~~이여라
    else if문: 만일 ~~~아닐 때 ~~~한다면 ~~~이여라
    if(money>3000){console.log('김밥을산다')}
    else if(money>1000){console.log('삼각김밥을산다')}
    else{console.log('굶는다')}
    (돈이 3000원보다 많으면 김밥을 먹을것이고, 아니라면 굶을것이다)
  • 반복문 - 구구단처럼 일정한 공식을 반복 출력 for.while
    for(let i = 0; i<9; i++){ console.log(i)}
    (i를 0부터 9미만까지 반복해서 콘솔 로그에 출력해라)
