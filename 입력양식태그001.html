입력양식태그란?

입력양식은 사용자의 정보를 입력받는 요소
예를 들어 로그인/회원가입/검색 등을 들 수 있다

form 태그 내부에 입력양식태그들을 넣어서 만든다
form 태그는 전달받는 곳에 형식을 기재해야한다

  • method="get" - 직접 전달 형식(당근마켓)
  • method="post" - 비밀 전달 형식(등기우편)
form 태그 외부에 입력양식 태그들을 기재할 수는 있지만, 전송할 수는 없다

입력양식종류

태그명 속성명(type="") 설명
form 특성: 보이지 않음/block 요소 입력양식의 시작과 끝
input text 글자입력양식
password 비밀번호양식설정
submit 제출버튼(확인)
reset 초기화버튼(취소)
file 내컴퓨터에 있는 파일 입력생성
search 검색박스로 텍스트 입력시 X버튼이 나타남
button 버튼생성
checkbox 사각형의 선택박스(다중선택)
radio 라디오버튼생성(단일선택)
같은 이름이라는 표식을 속성으로 꼭 표시하기 name="동일이름"
email 입력양식에 맞게 작성하지 않을경우 오류발생
number 숫자입력방식- 다양한 속성과 함께 작성 가능
color 컬러를 선택할 수 있는 기능
range 슬라이드바- 숫자범위로 나타남, 다양한 속성과 함께 작성 가능
datetime,month,week,date,time datetime(국제표준시간-년/월/일/시간, datetime-local로 인식 가능)
month(년/월),week(년/주),date(년/월/일),time(시간)
-------------------------------------type 외에 속성들----------------------------------------
required="required" 필수작성표시
placeholder="도움글" 입력양식의 힌트가 되는 글을 보여줌, 내용을 쓰기시작하면 지워짐
autofocus="autofocus" 입력필드에 커서가 자동으로 생성 - 내용작성유도(하지만 화면이 바로 이동함)
min="최소값" / max="최대값" / step="건너뛰기" 수가 들어가야 하는 모든 속성에 사용가능
value="눈에보이는글씨" 외부에 보이는 글씨 변경
textarea cols="가로크기값(글씨가 들어가는 최대수)" rows="세로값 (줄바꿈의 갯수)" 여러행의 글자입력 양식 text와 크기값이 같지 않음
select multipul="multipul" 선택 양식 생성
option 옵션 생성
optgroup label="그룹명" 옵션선택의 그룹지정으로 선택되지는 않음
fieldset 텍스트 그룹 양식 자동으로 선으로 묶어줌
legend fieldset의 타이틀 선의 내부에 작성됨
label 태그 라는 정의 태그로 form태그 내부의 결합하고자 하는 속성과 함께 사용된다(인라인 요소로 특별한 성질은 없음)
