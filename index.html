html 학습서

html/css/javascript의 학습내용을 첨부한다
  • html
  • css
  • javascript

html이란? Hyper Text Markup Language의 약자로 웹페이지를 읽을 수 있게 만들어주는 언어이다

  • 제목태그와 글자태그
  • 텍스트태그
  • div태그/span태그
  • img태그/a태그
  • ol태그/ul태그
  • table태그/tr태그/td태그
  • 시맨틱태그의 종류
  • form태그

css란? Cascading Style Sheet의 약자로 html 페이지 스타일을 지정해주는 스타일 시트이다

  • css선택자
  • css글자속성
  • css박스속성
  • css단위

Javascript는 html 페이지에서 사용자의 반응 등을 처리하는 스크립트 작성 언어이다
