박스속성

웹 페이지 레이아웃 구성시 가장 중요

  • margin- 여백
  • padding- 영역
  • width, height- 크기

작성법

  • 전체적용- 50px 하나의 숫자로 4방향 모두 적용
  • 각자적용- 위 우측 아래 좌측 / 모두 단위와 함께 작성, 중간에 쉼표 쓰지 않음
    위아래 좌우 두개로 적용 / 첫번째 숫자- 위아래 적용, 두번째 숫자- 좌우 적용
  • 부분적용
    • margin-left: 왼쪽여백
    • margin-right: 우측여백
    • margin-top: 위쪽여백
    • margin-bottom: 아래쪽여백
    • padding-left: 왼쪽영역
    • padding-right: 우측영역
    • padding-top: 위쪽영역
    • padding-bottom: 아래쪽영역

width: 가로크기값
width+(padding+border)*2

height: 세로크기값
height+(padding+border)*2

border

박스의 테두리를 말하며, 가로, 세로 값에 영향을 준다.
색상, 굵기, 모양을 한번에 모두 표기시 보인다.

작성법
border: black 2px solid (순서와 관계 없음)
4면 모두 한번에 나오거나 지정할 수 있음 border-left,right,top,bottom

선의 종류
bashed: 긴점선, dotted: 짧은점선, double: 두줄선, none:없음
액자선: groove, ridge, inset, outset

배경

배경색, 배경이미지, 배경반복, 배경위치, 배경고정 등을 설정한다.

  • background-color: 색깔이름, RGB, RGBA로 표기 / transparent: 투명색
  • background-image: url("위치주소기재")
  • background-repeat: repeat(가로, 세로 모두 반복), repeat-x(가로 반복), repeat-y(세로 반복), no-repeat(반복 없음)
  • background-position: 가로위치, 세로위치, 생략시 기본값(left, top)
    숫자로 지정시 왼쪽 상단 기준에서 시작됨 가로숫자, 세로숫자
    위치 명으로 기재 가능
  • background-attachment: 배경이미지를 고정시킬지 스크롤할지 결정
    scroll, fixed
  • background-size: 배경 이미지의 크기 설정
    • 100%: 화면을 비율 상관없이 꽉 채움(contain/cover중 한개랑 똑같아짐)
    • contain: 배경 이미지를 잘리는 부분 관계없이 이미지 우선으로 보임
    • cover: 배경 이미지와 관계없이 화면을 꽉 채움

둥근사각형

border-radius: 값을 적용하여 텍스트 필드나 div 등의 요소를 원, 반원, 1/4조각 등 다양한 형태를 제작할 수 있다.

작성법
border-radius: 30px; / border-radius: 10px 20px 30px 40px;
