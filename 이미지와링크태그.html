이미지와 링크 태그

이미지 태그

웹문서에 이미지를 불러오는 요소
인라인 태그로 줄바꿈이 되지 않음
닫힘 태그 요소 없음
이미지 태그와 함께 사용하는 필수 속성 태그가 존재함

이미지 태그 속성들

  • src="이미지 경로 지정"
  • width="가로 크기 지정(px단위 자동 설정)"
  • height="세로 크기 지정(px단위 자동 설정)"
  • alt="대체 텍스트 속성"

alt 속성은 웹 접근성을 사용할 때 필수 작성 요소

예제

불특정한 사진을 공간을 채우기위해 사용하는 사이트
https://picsum.photos/200/300 - 가로200*세로300 사이즈의 사진 등록
https://picsum.photos/ 뒤에 아무것도 없으면 크기를 몰라 사진을 안 보낸다



링크태그

웹 문서에서 다른 페이지나 웹 페이지 내부의 특정 위치로 이동할 때 사용하는 anchor(앵커) 태그 -a태그
닫힘 태그와 함께 사용하고 줄바꿈이 없는 인라인 속성이다.

앵커 태그 속성

  • href="위치 경로 주소 작성"
  • target="_blank" (새창열기)

예제

  • 네이버 바로가기
  • 네이버 새창열기
