선택자의종류

html 파일 css를 통해 꾸밀 수 있고 꾸미기 위해서는 지정된 선택을 해야한다

작성법

선택자{속성명 : 속성값; }

선택자종류 작성방법 설명
전체선택자 * html 내부의 모든 태그를 불러서 설정한다
태그선택자 태그명 html 내부의 불려진 모든 태그를 설정한다
아이디선택자 #아이디이름 속성값 ID의 이름을 불러서 특정 ID만 설정
클래스선택자 .클래스명 속성값 class의 이름을 불러서 특정 클래스만 설정
id와 class의 차이점:
id는 한 페이지에 한번만 사용, 중복사용불가 한 번 사용한 아이디는 두 번 사용 안 됨
class는 중복 적용 가능, 한 클래스에 여러 클래스명 사용가능
클래스명 한칸 띄어쓰기로 여러 클래스명 부여 가능
중복선택자 , 여러개의 선택자를 나열해서 사용할 수 있다.
중복선택자 붙여쓰기(태그.클래스명) 지정된 태그중 해당 클래스명만 부름
후손선택자 선택자 선택자 부모자 내부에 있는 요소를 모두 부른다.
자손선택자 부모자>자손자 다이렉트 자손자에 한해서 설정 가능
