요소선택

DOM(Document Object Model): HTML같은 문서를 선택,생성,삭제 등을 활용

직접선택메서드

메서드이름 설명
getElementById('') HTML ID요소를 가져옴
getElementsClassName('') HTML Class요소들을 가져옴
getElementsTagName('') HTML Tag요소들을 가져옴
querySelector('') 선택된 요소중 첫번째 요소만 가져옴
Tag를 제외하고 Class/ID를 부를 때는 CSS와 동일하게 부름
querySelectorAll('') 선택된 모든 요소중 요소만 가져옴
Tag를 제외하고 Class/ID를 부를 때는 CSS와 동일하게 부름

Event

선택자 혹은 도큐먼트상에서 행동을 가했을 때 발생하는 현상

이벤트 설명
click 마우스를 클릭했을 때 이벤트가 발생한다.
mouseover 마우스가 일정 공간에 올라왔을 때 이벤트가 발생한다.
mousemove 마우스가 움직였을 때 이벤트가 발생한다.
mouseenter 마우스가 일정 공간에 들어왔을 때 이벤트가 발생한다.
mouseleave 마우스가 일정 공간에서 떠났을 때 이벤트가 발생한다.
submit submit 버튼을 눌렀을 때 이벤트가 발생한다.
load 페이지를 열었을 때 이벤트가 발생한다.
resize 도큐먼트 사이즈가 변경되었을 때 이벤트가 발생한다.
scroll 스크롤바가 움직였을 때 이벤트가 발생한다.
