요소선택

DOM(Document Object Model): HTML같은 문서를 선택,생성,삭제 등을 활용

직접선택메서드

메서드이름 설명
getElementById('') HTML ID요소를 가져옴
getElementsClassName('') HTML Class요소들을 가져옴
getElementsTagName('') HTML Tag요소들을 가져옴
querySelector('') 선택된 요소중 첫번째 요소만 가져옴
Tag를 제외하고 Class/ID를 부를 때는 CSS와 동일하게 부름
querySelectorAll('') 선택된 모든 요소중 요소만 가져옴
Tag를 제외하고 Class/ID를 부를 때는 CSS와 동일하게 부름
